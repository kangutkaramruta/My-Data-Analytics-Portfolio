# MarketCo Marketing Analytics Project

## Overview
Customer segmentation and marketing performance analysis focused on membership performance, communication effectiveness, demographic spend drivers, and retention strategy.

## Key Focus Areas
- Gold vs Non-Gold membership performance
- Spend analysis by age, state, and product category
- Communication pattern analysis (inbound vs outbound)
- Web analytics data recommendations (A/B testing, sentiment analysis)
